## English
### Overview
OneStopShop is a complete marketplace-style web application using Next.js App Router, focused on storefront performance and operational seller tools.

### Core Features
- Storefront with product discovery and quick view
- Checkout flow with Stripe integration
- Seller area with product and order management
- Authentication with Clerk
- File uploads with UploadThing
- Database and schema tooling with Drizzle ORM

### Tech Stack
- Next.js 14, React 18, TypeScript
- Tailwind CSS, Radix UI
- Stripe, Clerk, UploadThing
- Drizzle ORM, MySQL
- Storybook for component documentation

### Getting Started
```bash
npm install
npm run dev
```
